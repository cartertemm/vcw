# Falsehoods About Time with Explanations

Explanations for all 112 things that programmers falsely believe about time. Each "falsehood" represents an assumption that seems reasonable but fails in real-world scenarios. Based on the gist by Tim Visee: [Falsehoods programmers believe about time](https://gist.github.com/timvisee/fcda9bbdff88d45cc9061606b4b923ca).
